#Diff maker
## Usage  
`gendiff -h` - to see the help

Usage: `gendiff [options] <firstConfig> <secondConfig>`
### Options  

    -h, --help           output usage information
    -V, --version        output the version number
    -f, --format [type]  Output format
       
# Installation
`npm install @vikzh/diffgen`
